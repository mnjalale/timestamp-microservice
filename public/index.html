Timestamp microservice

How to use it:

  1. Pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example January, 1, 2016)
  2. If it does, it returns both the Unix timestamp and the natural language form of that date
  3. If it does not contain a date or Unix timestamp, it returns null for those properties

Example usage:

https://timestamp-microservice-mnjalale.c9users.io/March%2015,%202015
https://timestamp-microservice-mnjalale.c9users.io/1426377600

Example output:

{ "unix": 1426377600, "natural": "March 15, 2015" }
